# DreamRoam - Smart Travel Booking Website

A modern, responsive single-page travel booking website with frosted glass dark mode, interactive carousels, and smooth animations built with HTML, CSS, and JavaScript.

## Features

- Dark Mode Toggle: Frosted glass blur effects with smooth theme switching
- Fully Responsive: Mobile-first design works on all screen sizes
- Swiper Carousels: Popular destinations and client testimonials sliders
- Interactive Search: Trip planner with destination, date, and budget inputs
- Glassmorphism UI: Modern translucent card effects and hover animations

## How It Works

The website uses a single-page layout with smooth scrolling navigation. Users can toggle between light/dark themes, browse popular destinations through touch-enabled carousels, read client testimonials, and plan trips using the interactive search form. All interactions are handled with vanilla JavaScript and modern CSS.

### Technology Used
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Libraries: Swiper.js, Remixicon icons
- CSS Features: Grid, Flexbox, Backdrop-filter, CSS Custom Properties

## Getting Started

### Prerequisites
To view this website, you need:
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Running the Website
1. Clone the repository (or download the code) and navigate to the project directory
2. Open `index.html` in any web browser:
   ```bash
   # No compilation needed - just open!
   double-click index.html
   # or
   live-server (VS Code extension)
   ```
